# robot_miniProject

โปรเจกต์นี้เป็นระบบหยิบและวาง (Pick and Place) โดยใช้แขนกลหุ่นยนต์ (น่าจะเป็น Dobot MG400) ร่วมกับการประมวลผลภาพ (Computer Vision) ผ่านกล้องเว็บแคม ระบบประกอบด้วยสองส่วนหลัก:

โปรแกรมบน PC (Python): ทำหน้าที่ประมวลผลภาพจากกล้อง, ค้นหาวัตถุตามสีที่กำหนด, แปลงพิกัดจากกล้องเป็นพิกัดของหุ่นยนต์ และส่งคำสั่งไปให้หุ่นยนต์ผ่าน TCP/IP

โปรแกรมบนหุ่นยนต์ (Lua): ทำหน้าที่เป็น TCP Server รอรับคำสั่งพิกัดจาก PC และสั่งให้หุ่นยนต์เคลื่อนที่ไปยังจุดต่างๆ เพื่อหยิบและวางวัตถุ
